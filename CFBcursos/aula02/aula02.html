Aula 02 - Modo Estrito

Javascript em Modo Estrito, você sabe o que é? - Curso de Javascript Moderno - Aula 02

Hoje em nosso curso de Javascript Completo e Profissional vamos aprender o que é o modo estrito, usando o comando "use strict". O modo estrito ou strict mode faz com que a programação seja mais limpa e funcional.

O modo estrito, também conhecido como "strict mode" em inglês, é um recurso do JavaScript que foi introduzido na versão ECMAScript 5 (ES5) da linguagem. Ele é usado para impor uma forma mais restrita de interpretação e execução do código JavaScript, aplicando um conjunto mais rigoroso de regras e restrições.

Ao ativar o modo estrito em um programa JavaScript, algumas mudanças importantes ocorrem no comportamento padrão da linguagem. Essas mudanças têm o objetivo de eliminar ou modificar certas características consideradas propensas a erros ou que podem levar a comportamentos inesperados.

Algumas das características do modo estrito do JavaScript incluem:

  • Declaração explícita de variáveis: No modo estrito, é necessário declarar as variáveis com a palavra-chave "var", "let" ou "const" antes de usá-las. Caso contrário, ocorrerá um erro.
  • Erros para atribuição a propriedades somente leitura: No modo estrito, atribuir um valor a uma propriedade somente leitura ou a uma variável não modificável (constante) lançará um erro.
  • Deletar variáveis, funções e argumentos não é permitido: No modo estrito, a ação de deletar variáveis, funções ou argumentos de função não é permitida e lançará um erro.
  • Não é permitido usar palavras reservadas como nomes de variáveis: No modo estrito, o uso de palavras reservadas como identificadores de variáveis lançará um erro.
  • Parâmetros de função com nomes duplicados não são permitidos: No modo estrito, uma função com parâmetros de nomes duplicados lançará um erro.

Essas são apenas algumas das alterações introduzidas pelo modo estrito. Ativar o modo estrito é uma prática recomendada para escrever código JavaScript mais seguro e robusto, ajudando a evitar erros comuns e melhorar a legibilidade e manutenção do código. Para ativar o modo estrito, basta adicionar a seguinte linha ao início do seu script: "use strict";

É importante observar que o modo estrito não é ativado automaticamente em programas JavaScript, sendo necessário habilitá-lo explicitamente. Isso ocorre para garantir a compatibilidade com versões anteriores do JavaScript, permitindo que o código escrito em versões anteriores continue funcionando conforme o esperado.
