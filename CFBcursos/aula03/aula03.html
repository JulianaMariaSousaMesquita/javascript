Aula 03 - LET, VAR e CONST

Diferença entre LET, VAR e CONST em Javascript - Curso de Javascript Moderno - Aula 03

Hoje em nosso curso de Javascript Completo e Profissional vamos aprender sobre as diferenças das declarações usando let, var e const em Javascript.

Em JavaScript, let, var e const são palavras-chave utilizadas para declarar variáveis, mas possuem algumas diferenças importantes.

  • var é a forma mais antiga de declarar variáveis em JavaScript e possui escopo de função. Isso significa que uma variável declarada com var é válida dentro da função onde foi declarada ou, se não estiver dentro de uma função, é uma variável global acessível em todo o código. Além disso, var permite a redeclaração e a reatribuição de valores para a mesma variável dentro do mesmo escopo.
  • let foi introduzido no ECMAScript 6 (ES6) e possui escopo de bloco. Isso significa que uma variável declarada com let é válida apenas dentro do bloco onde foi declarada, como um bloco de função, um bloco if, um loop for, etc. Além disso, let não permite a redeclaração da mesma variável no mesmo escopo, mas permite a reatribuição de valores.
  • const também foi introduzido no ES6 e possui escopo de bloco assim como o let. A diferença é que uma variável declarada com const é uma constante, ou seja, seu valor não pode ser alterado após a atribuição inicial. Ela deve ser inicializada no momento da declaração e não pode ser reatribuída posteriormente.

É importante ressaltar que o escopo de bloco do let e const pode ajudar a evitar problemas como vazamento de variáveis e redeclaração acidental, tornando o código mais seguro e mais fácil de entender. Portanto, é recomendado o uso de let e const em vez de var na maioria dos casos, a menos que haja uma necessidade específica de utilizar o comportamento do var.
